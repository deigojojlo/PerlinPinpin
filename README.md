l'objectif de ce projet est d’etre capable de generer un monde en 3d de manière procedurale.

Le but ce rapproche du jeu video minecraft,
qui propose 
- un monde, divisé en biomes
- un biome est un groupe de chunk
- les biomes ont une transition fluide entre eux
- des rivières, lacs et océan
- du relief, montagne, plaine....


Il faut etre capable à partir d’une seed, d’obtenir exactement le même monde 
un monde est potentiellement infini
pour l’instant, on ne remplit pas le monde en hauteur


Le projet est en cours. 

pour l’instant, j’ai un bruit de perlin, qui est la base de tout.
En particulier, j’ai la génération des biomes qui est fonctionnelle
